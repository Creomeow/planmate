# PlanMate

PlanMate is a mobile app built with Expo and React Native that helps users plan and manage their events.

## Features

- User authentication (login/register)
- Profile management
- Event exploration
- User preferences
- Modern UI with React Native Paper

## Setup

1. Clone the repository
```bash
git clone <your-repo-url>
cd PlanMate
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npx expo start
```

4. Run on your device
- Scan the QR code with Expo Go (Android)
- Scan the QR code with Camera app (iOS)

## Technologies Used

- Expo
- React Native
- React Native Paper
- Expo Router
- AsyncStorage
